# Storage Review Pack

This folder contains the cleaned-up review materials for the 2025 FAST, OSDI, and SOSP storage-related papers.

Recommended review order:

1. `01-top5.md`
   Start here if you want the shortest high-signal reading list.
2. `02-detailed-report.md`
   Read this next for the fuller technical report and per-paper commentary.
3. `03-paper-table.csv`
   Use this as the compact reference table for sorting, filtering, or spreadsheet import.

File guide:

- `01-top5.md`
  The five papers most worth tracking for the storage community.
- `02-detailed-report.md`
  A longer report organized by conference and technology theme.
- `03-paper-table.csv`
  A flat table of the filtered papers with conference, category, authors, and summary.

Scope notes:

- No paper in this set explicitly mentions `Ceph` in the public title or abstract.
- The clearest explicit `LSM-tree` paper in this set is `AegonKV`.
- The `Mantle` summary in the detailed report remains partially inference-based because the SOSP accepted-papers page exposes title and authors but not a public abstract.
